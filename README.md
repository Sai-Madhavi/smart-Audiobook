# 🎧 Smart AI Audiobook Reader

A simple and lightweight application that converts PDF and DOCX documents into speech using an offline Text-to-Speech engine.

## 🚀 Features
- Upload PDF and DOCX files
- Converts text into speech
- Reads document page by page
- Remembers last read page (resume feature)
- 15-minute snooze (auto stop)

## 🛠 Technologies Used
- Python
- Streamlit
- PyPDF2
- python-docx
- pyttsx3

## ▶ How to Run

1. Install dependencies: pip install streamlit PyPDF2 python-docx pyttsx3

2. Run the app: streamlit run app.py

## 📌 Note
This project uses an offline Text-to-Speech engine (pyttsx3), so it runs locally and cannot be deployed on GitHub Pages.
